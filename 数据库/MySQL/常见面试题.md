

# 写SQL的题目

## 基础

1. 查询分数排在第三名的同学的成绩： 主要是用limit带两个参数去确定第三位。
   1. 第一个参数表示记录的起始位置(第一个记 录的位置是 0)，第二个参数表示返回几条记录。
   2. “LIMIT 2,1”就表示从第 3 条记 录开始，返回 1 条记录。
2. Having操作的理解： 
   - 原理： HAVING 则需要跟分组关键字
GROUP BY 一起使用，通过对分组字段或分组计算函数进行限定，来筛选结果。
   - 例题： 找出带学生超过2个的老师：
```sql
SELECT teacherid
FROM demo.teach
WHERE teacherid IS NOT NULL
GROUP BY teacherid
HAVING COUNT(*)>=2
```
